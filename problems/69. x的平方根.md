### 69. x的平方根

#### 题目地址

https://leetcode-cn.com/problems/sqrtx/

#### 题目标签

数学	二分查找

#### 题目描述

实现 `int sqrt(int x) `函数。

计算并返回 x 的平方根，其中 x 是非负整数。

由于返回类型是整数，结果只保留整数的部分，小数部分将被舍去。

**示例 1:**

```
输入: 4
输出: 2
```

**示例 2:**

```
输入: 8
输出: 2
说明: 8 的平方根是 2.82842..., 
     由于返回类型是整数，小数部分将被舍去。
```

#### 思路

太暴力了...从1开始遍历到`i`的平方大于x为止，`i-1`即为最后的结果....

#### 题解

```c++
class Solution {
public:
    int mySqrt(int x) {
        long long i=0;
        for(i=1; i*i<=x; i++)
        {
            if(i*i>x) break;
        }
        return i-1;
    }
};
```
